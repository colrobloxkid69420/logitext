================================================================================
DEBUGGING & TROUBLESHOOTING
================================================================================

If you encounter issues:

1. Module not found errors:
   - Clear node_modules: rm -rf node_modules package-lock.json
   - Reinstall: npm install

2. Build fails:
   - Check Node.js version: node -v (should be 18+)
   - Clear .vite cache: rm -rf .vite
   - Try: npm run build

3. Port 5173 already in use:
   - vite.config.ts sets port, can be changed
   - Or manually run: npm run dev -- --port 3000

4. TypeScript errors:
   - Check tsconfig.json is correct
   - Verify path aliases in vite.config.ts
   - Look for import path issues

5. Tailwind CSS not applying:
   - Verify @import "tailwindcss" in styles.css
   - Check tailwind.config.js has correct content paths
   - Clear node_modules and reinstall
